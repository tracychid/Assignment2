# Assignment #2 Repository

This repository includes the simulated data for Assignment #2. Fork this repository and add your analysis as described in the canvas assignment.
The csv file for `cohort` in the `raw-data` folder includes 5,000 observations with variables `smoke`, `female`, `age`, `cardiac`, and `cost`.



Summary of Findings:

Descriptive analysis showcased 5.5% prevalence of cardiac event (N = 275) across the 5,000 study observations with 
12% (N = 33) females and 88% (N = 242) males experiencing cardiac events. Individuals who experienced a 
cardiac event had an average age of 46.4 (SD = 16.2) and an average cost of 10,200 (SD = 427). Using logistic
regression, our main study question revealed a 1% increase in the risk for cardiac event with a one-unit increase
in cost. Adjusted odds ratios showed smoking as protective (OR = 0.05, CI = 0.03, 0.09), but this may be driven
to the imbalance in smoking and rates of cardiac events in females compared to males who make up 88% of
cardiac events in our study.
